MdDS Web Application

Welcome

This web application helps doctors and researchers treat patients with MdDS (Mal de DÃ©barquement Syndrome). A common treatment involves the patient watching stripes moving across a screen. This web application provides this treatment, and works on any modern browser or mobile device.

It can also be used offline if downloaded from here. After downloading, unzip the downloaded file to a location on your computer and open index.html in your web browser. This code is open-source, and available to everyone on GitHub.

System Requirements

On Windows and Mac, this application supports the latest versions of the following browsers: Google Chrome, Mozilla Firefox, Internet Explorer 10+, Safari, Opera, and Microsoft Edge. Mobile device support varies by device and vendor (the latest versions of major mobile browsers are typically supported). This application works in browsers which support HTML5 Canvas and requestAnimationFrame().
